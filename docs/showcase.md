# Screenshot / GIF Showcase

This guide helps you prepare release-ready visual assets for GitHub.

## What to capture

- Main desktop (top bar + dock)
- Wofi launcher
- Settings menu
- Wallpaper menu (local/search/favorites/history)
- Theme switch examples (neon-purple, crimson-night, mono-dark)

## Recommended assets

- `assets/showcase/desktop-main.png`
- `assets/showcase/settings-menu.png`
- `assets/showcase/wallpaper-menu.png`
- `assets/showcase/theme-neon-purple.png`
- `assets/showcase/theme-crimson-night.png`
- `assets/showcase/theme-mono-dark.png`
- `assets/showcase/demo.gif` (optional)

## Capture tips

- Use same resolution and scale for consistency
- Hide sensitive data before capture
- Keep GIF under 15 MB for README
