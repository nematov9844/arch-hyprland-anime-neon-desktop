# ⌨️ Keyboard Shortcuts

Default keybindings for the system.

Repo: [nematov9844/arch-hyprland-anime-neon-desktop](https://github.com/nematov9844/arch-hyprland-anime-neon-desktop)

---

## 🪟 Window Management

| Action            | Shortcut        |
| ----------------- | --------------- |
| Open terminal     | `SUPER + ENTER` |
| Close window      | `SUPER + Q`     |
| Toggle fullscreen | `SUPER + F`     |
| Toggle floating   | `SUPER + V`     |

---

## 🧭 Navigation

| Action      | Shortcut                     |
| ----------- | ---------------------------- |
| Move focus  | `SUPER + Arrow keys`         |
| Move window | `SUPER + SHIFT + Arrow keys` |

---

## 🧱 Workspaces

| Action                   | Shortcut              |
| ------------------------ | --------------------- |
| Switch workspace         | `SUPER + 1-0`         |
| Move window to workspace | `SUPER + SHIFT + 1-0` |

---

## 🔍 System

| Action      | Shortcut            |
| ----------- | ------------------- |
| Launcher    | `SUPER + D`         |
| Power menu  | `SUPER + SHIFT + E` |
| Lock screen | `SUPER + L`         |
| Settings Center | `SUPER + W` |
| Settings Center | `SUPER + ,` |
| Random wallpaper | `SUPER + SHIFT + W` |

---

## 🔁 Legacy Compatibility

Legacy Wofi menus (`settings-menu.sh`, `wallpaper-menu.sh`) are still installed as compatibility fallback,
but they now redirect to `settings-center`.

## 📸 Screenshots

| Action          | Shortcut            |
| --------------- | ------------------- |
| Area screenshot | `SUPER + SHIFT + S` |
| Full screenshot | `PRINT`             |

---

## 🔊 Volume

| Action      | Shortcut               |
| ----------- | ---------------------- |
| Volume up   | `XF86AudioRaiseVolume` |
| Volume down | `XF86AudioLowerVolume` |
| Mute        | `XF86AudioMute`        |

---

## 💡 Brightness

| Action          | Shortcut                |
| --------------- | ----------------------- |
| Brightness up   | `XF86MonBrightnessUp`   |
| Brightness down | `XF86MonBrightnessDown` |
