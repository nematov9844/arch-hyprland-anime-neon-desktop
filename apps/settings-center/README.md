# Settings Center

`settings-center` - Arch Hyprland Anime Neon Desktop uchun markaziy sozlamalar UI ilovasi.

Settings Center is now the main control UI. Legacy Wofi menus are kept as compatibility fallback.

## Tuzilma

- `src/app.py`: kirish nuqtasi
- `src/views/`: asosiy ekranlar
- `src/widgets/`: qayta ishlatiladigan UI komponentlar
- `src/services/`: biznes logika va fayl/system integratsiyasi
- `src/utils/`: yordamchi utility funksiyalar
- `assets/`: ikonka, preview, statik fayllar

## Ishga tushirish (placeholder)

```bash
python3 apps/settings-center/src/app.py
```
